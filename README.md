# 🛒 V Mart – Smart Grocery E-Commerce Platform

V Mart is a full-stack smart grocery e-commerce platform built to deliver a secure, personalized, and seamless online shopping experience.

The platform combines core e-commerce functionality with behavior-based product recommendations, voice search, smart shopping lists, personalized offers, real-time order tracking, and multilingual support.

Built with the MERN stack, V Mart focuses on scalable architecture, secure backend APIs, real-time communication, and intelligent shopping experiences.

## 📖 About V Mart

V Mart is designed as a modern grocery shopping platform where users can discover products, manage their cart, create reusable shopping lists, place orders, and track their purchases in real time.

The platform goes beyond a traditional e-commerce workflow by analyzing shopping activity and purchase patterns to improve product discovery and personalize the user experience.

User interactions such as product views, search activity, cart behavior, and previous orders are processed by an internal personalization engine to generate relevant product recommendations, smart shopping suggestions, and contextual offers.

---

## 🧠 Personalized Product Recommendation Engine

V Mart includes a custom-built recommendation engine that analyzes user shopping behavior and product interaction patterns.

The recommendation system considers:

* Product views
* Search activity
* Frequently explored categories
* Cart additions
* Previous orders
* Purchase frequency
* Recent shopping activity
* Product popularity

A weighted scoring approach ranks products based on their relevance to each user's shopping behavior.

### Recommendation Sections

* Recommended For You
* You May Also Like
* Based on Your Recent Activity
* Similar Products
* Frequently Bought Together

For users with limited activity history, the platform dynamically displays products using actual shopping and order data, including popular and frequently purchased products.

Frequently Bought Together suggestions are generated by analyzing products commonly purchased in the same orders.

Similar products are discovered using attributes such as category, brand, and price range.

## 📍 Real-Time Order Tracking

V Mart provides real-time order status tracking using Socket.IO.

Users can monitor order progress without manually refreshing the application.

### Order Tracking Stages

* Order Placed
* Order Confirmed
* Processing
* Packed
* Out for Delivery
* Delivered
* Cancelled

A visual tracking timeline displays completed, current, and pending order stages.

When an authorized admin updates an order status:

1. The new status is validated.
2. The order is updated in MongoDB.
3. A real-time event is emitted.
4. The event is delivered to the authenticated user.
5. The tracking interface updates automatically.
6. An in-app notification is displayed.

### Real-Time Architecture Features

* Authenticated Socket.IO connections
* User-specific communication rooms
* Secure order events
* Automatic reconnection handling
* Latest order status synchronization
* Duplicate listener prevention
* Event listener cleanup

Users only receive updates related to their own orders.

## 🎁 Personalized Offers & Discount Engine

V Mart includes a rule-based personalization engine for generating relevant shopping offers.

The system evaluates:

* Previous purchases
* Frequently purchased categories
* Cart activity
* Order frequency
* Last order activity
* Shopping patterns

### Personalized Offer Types

* Category-based discounts
* Cart completion offers
* Returning user offers
* Inactive user offers
* Frequent shopper offers

Offer eligibility and final discount calculations are validated by the backend.

The system protects against:

* Expired offer usage
* Unauthorized offer access
* Discount manipulation
* Usage limit bypass
* Invalid offer stacking
* Reuse of restricted offers
* Ineligible product discounts

Product prices and eligible cart totals are recalculated using current database values before applying an offer.

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone <your-repository-url>
```

### 2. Navigate to the Project

```bash
cd V-Mart
```

### 3. Install Frontend Dependencies

```bash
cd frontend
npm install
```

### 4. Install Backend Dependencies

```bash
cd ../backend
npm install
```

---

## 🔐 Environment Configuration

Configure the required environment variables according to the project setup.

### Example Backend Configuration

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
PORT=5000
CLIENT_URL=your_frontend_url
```

Additional environment variables may be required depending on the existing authentication, media storage, payment, or deployment configuration.

Never commit real credentials, tokens, passwords, or secret values to the repository.

## ▶️ Running the Application

### Start the Backend

```bash

```

### Start the Frontend

Open another terminal:


Open the local development URL displayed by the frontend development server.
